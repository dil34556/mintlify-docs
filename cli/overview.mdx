## About the CLI

The Telminator CLI is a lightweight, powerful command-line interface that puts the full capabilities of A2A (Agent-to-Agent) communication at your fingertips.

<Note>
**A2AClient in Action:** The CLI is a host application that demonstrates the capabilities of an `A2AClient`. It reads a server's `AgentCard` and enables you to collaborate with remote agents through simple text commands. All responses from the A2A server are displayed directly in your console.
</Note>
