# Telminator Documentation

Welcome to the Telminator documentation. Learn how to integrate with Telminator agents and build powerful AI-driven applications.

## What is Telminator?

Telminator is a platform for building and deploying AI agents that can communicate with each other and integrate with external systems through standardized protocols.

## Getting Started

Choose your integration path:

<Steps>
  <Step title="Install CLI">
    Set up the Telminator CLI for quick agent testing
    
    [View Installation Guide →](/docs/cli/installation)
  </Step>
  
  <Step title="Get API Key">
    Generate your API key from the dashboard
    
    [Get Your Key →](https://app.telminator.com/#api-keys)
  </Step>
  
  <Step title="Choose Integration">
    Pick your preferred integration method (CLI, Python, MCP)
  </Step>
  
  <Step title="Start Building">
    Follow our guides to integrate Telminator into your application
  </Step>
</Steps>

## Features

### CLI Tool
- Interactive command-line interface
- Connect to any A2A agent
- Real-time streaming responses
- File attachment support

### Python Integration
- Complete A2A client implementation
- Streaming support
- File handling
- Task management

### MCP Support
- Standard Model Context Protocol
- Tool discovery and execution
- JSON-RPC interface
- Easy integration with AI applications
