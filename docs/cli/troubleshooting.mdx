## Troubleshooting

### Authentication Errors

If you see an authentication error:

<Note type="warning">
**Auth Error:** Invalid credentials or service issue. Check your API key.
</Note>

This usually means:

- Your API key is incorrect or expired
- The agent URL is wrong
- The agent service is not running

**Solution:** Generate a new API key from your Telminator dashboard and restart the CLI.
